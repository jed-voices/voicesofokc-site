## Replace before launch
1. `CNAME` with your real domain
2. `robots.txt` and `sitemap.xml` with your real domain
3. Spotify and Apple Podcast links (already added) in `index.html` and episode pages
4. Contact email in the footer
5. Optional additional episode pages and guest thumbnails

## GitHub Pages deploy
1. Create a new repository on GitHub, such as `voicesofokc-site`
2. Upload all files from this folder
3. In GitHub, open **Settings → Pages**
4. Set source to **Deploy from a branch**
5. Choose **main** branch and **/(root)**
6. Add your custom domain
7. Update DNS with your domain registrar
